goctl介绍

goctl 读作 go control，不要读成 go C-T-L。goctl 的意思是不要被代码控制，而是要去控制它。其中的 go 不是指 golang。在设计 goctl 之初，我就希望通过 她 来解放我们的双手👈

名称功能示例

new快速创建一个api服务goctl api new user
formatapi格式化，vscode使用

-iu是否自动更新goctl
-stdin是否从标准输入读取数据

-api 指定api文件源		goctl api validate -api user.api
doc生成doc markdown
-dir指定目录		goctl api doc -dir user

-dir指定生成代码目录

名称功能示例
-name服务名称

-replicas指定副本数
-revisions指定保留发布记录数
-port指定服务端口
-nodePort指定服务对外暴露端口
-minReplicas指定最小副本数
-maxReplicas指定最大副本数
